  • Departure Rate

    The Departure Rate project was built for a client to measure the amount of time a car was at a drive-through window. The time is displayed on a TV using a web app that then sends the data to a database to be used for business metrics. This project was built using React, Flask, SQLAlchemy, HTML, CSS, JavaScript, Python, a Raspberry Pi, and a HC-SR04 sensor. The site is not available to view due to it being used internally at a business.
